# Prompt Alternatives Gallery Research

This document lists the official galleries and visual resources for the prompt alternatives under analysis.

## 1. Oh-My-Posh (Rust-based, Cross-Platform)
*   **Gallery URL:** [Oh-My-Posh Themes](https://ohmyposh.dev/docs/themes)
*   **Description:** A massive collection of highly customizable themes.
*   **Look for:** Themes with "Northern Lights" colors (blues, teals, purples) like `catppuccin`, `dracula`, or `nord` variants.

## 2. Starship (Current Tool)
*   **Presets URL:** [Starship Presets](https://starship.rs/presets/)
*   **Configuration:** [Starship Configuration](https://starship.rs/config/)
*   **Description:** The official presets show different styles (e.g., "Nerd Font Symbols", "Bracketed Segments") that can be achieved without changing the core tool.

## 3. Fish-Native Prompts
These run natively in Fish shell, often offering superior speed (async rendering).

*   **Tide:**
    *   **Repo/Screenshots:** [Tide GitHub](https://github.com/IlanCosman/tide)
    *   **Description:** The modern standard for Fish. Asynchronous, feature-rich, and includes a "Wizard" for easy configuration.
*   **Pure:**
    *   **Repo/Screenshots:** [Pure GitHub](https://github.com/pure-fish/pure)
    *   **Description:** Minimalist, fast, and elegant. Good if we want to reduce visual noise.
*   **Hydro:**
    *   **Repo/Screenshots:** [Hydro GitHub](https://github.com/jorgebucaran/hydro)
    *   **Description:** Ultra-fast, minimal, lag-free.
